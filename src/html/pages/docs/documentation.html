About the template

The Railsblocks Theme template is divided in two main parts: The Admin Dashboard that contains all components for the distributor and reseller dashboards, and the Store that contains all components for the shopping process.

All html pages are listed under the src/html/ directory, and you can check how all of them look with the Railsblocks Theme.

All the stylesheets responsible for stylling these pages are found under the src/stylesheets/ directory.

Customizations

In case you want to make any customizations, we suggest creating separate files for it, since new versions of the theme are going to be released in the future. This makes easier for you to use the same custom file in other versions.

You can create a custom file for example:

stylesheets/
    ├── base/
        ├──dash/
        ├──others/
        ├──shop/ 
    ├── theme.less
    ├── custom.less

And then import it on theme.less:

@import "custom.less";

Admin Dashboard Theme

Main HTML elements

That is the dashboard for Distributor and Reseller. Theme files have almost unchanging structure consisting of:

  1. #wrapper main container of body elements.
  2. nav .navbar-static-side left navigation menu.
  3. #page-wrapper main container for page elements.
  4. nav .navbar-static-top top navigation with second menu.
  5. .page-heading container with page title and breadcrumb
  6. .wrapper-content main container for html elements.
  7. .footer main container for footer.

Layout structure

The base theme for the admin context covers 5 main sections of the layout and also some basic CSS that aligns all together. Those 5 main sections are:

  • Fixed left sidebar
  • Top navbar
  • Page title and breadcrumb
  • Main content
  • Fixed footer

Fixed left sidebar

The left sidebar contains the main menu. This main menu has all the main links to the other internal system pages.

Top navbar

The top navbar contains some user interaction elements such as: messages, notifications and also a profile menu.

Page title and breadcrumb

Contains only the page title and the page breadcrumbss.

Main content

The main part of the layout, the system pages will be loaded in this section.

Fixed footer

The footer is always sticky to the bottom of the page. In the footer, there is a tiny menu containing links such as "Help", "Feedback" and "Contact".

Store Theme

Main HTML elements

Most of the time the structure is as follows, although it might change in some pages:

  1. #shop main container for body elements.
  2. #header navigation menu.
  3. #content main container for page elements.
  4. #page-menu navigation sub-menu
  5. .page_title container with page title and breadcrumb
  6. #page_content main container for html elements.
  7. .big-footer main container for footer.

Layout structure

The base theme for the store context covers 4 main sections of the layout and also some CSS to align them all together. Those 4 main sections are:

  • Fixed header
  • Page title and breadcrumb
  • Main content
  • Fixed footer

Header

The header contains the main menu. Here we can find the main links for navigation on the system.

Page title and breadcrumb

Contains the page title, sometimes with a brief description, and the page breadcrumbs.

Main content

Theat is the main part of the layout, the content will be loaded in this section.

Footer

The footer is at the bottom of the page. There are many useful links on the footer, such as links for navigation and social media.

LESS Structure

The css structure should be always well organized in order to keep a clearly understanding of its elements, classes and also files.

All the LESS files are in the stylesheets directory and they are compiled into only one css file (public/stylesheets/global.css).

The file sctructure of the LESS files is:

stylesheets/
    ├── base/
        ├──dash/
        ├──others/
            ├──_helpers.less
            ├──_mixins.less
            ├──_variables.less
        ├──shop/  
    ├── theme.less

_helpers.less

Contains styllings that help all theme.

_mixins.less

This file contains all the stylesheets mixins the are used in other less files.

_variables.less

This file contains all the coloring styles and other variables useful to be used in the other less files.

shop/ & dash/

These directories contain all stylesheets responsible for the Store and Admin respectively. In order to maintain the organization we decided to break the theme in smallers files and different folders. Despite our theme is considerably big, it's very easy to find what you are looking for.

HTML Structure

As well as the stylesheets, the HTML strucuture should be always well organized in order to keep a clearly understanding of its structure.

The file sctructure of the HTML files is:

html/
    ├── layouts/
        ├── admin.html
        ├── shop.html
        ├── documentation.html
        ├── login.html
    ├── parts/
        ├── footer.html
        ├── header.html
        ├── navbar-top.html
        ├── page-nav.html
        ├── social-icons-font.html
        ├── soial-icons-svg.html
    ├── pages/
        ├── distributor/
        ├── reseller/
        ├── shop/
        ├── docs/

layouts/

Contains all layouts of the theme

parts/

Contains pieces of HTML which are common to other pages, such as headers and footers.

pages/

Contains all the pages of the theme, and is subdivided in:

  • distributor/

    Keeps all pages that the distributor finds on his dashboard.

  • reseller/

    Keeps all pages that the reseller finds on his dashboard.

  • shop/

    Keeps all pages that are found on the store.

  • docs/

    Keeps all the documentation.
